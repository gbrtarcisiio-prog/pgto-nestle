# Paleta de Cores Baseada no Logo Nestlé Sorvetes

A paleta de cores foi extraída do logo fornecido, focando em transmitir credibilidade, segurança e um estilo moderno, conforme as diretrizes.

| Nome da Cor | Uso Sugerido | Código Hex |
| :--- | :--- | :--- |
| **Primária (Nestlé Red)** | Botões de Ação (CTA), Destaques, Linhas de Ênfase | `#E31837` |
| **Secundária (Deep Blue)** | Fundo de Seções, Borda Principal, Elementos Institucionais | `#004A99` |
| **Terciária (Sky Blue)** | Gradientes, Elementos Visuais Suaves, Microinterações | `#00AEEF` |
| **Fundo (White)** | Fundo Principal da Página, Modais | `#FFFFFF` |
| **Texto (Dark Gray)** | Texto Principal, Ícones | `#333333` |
| **Neutro (Light Gray)** | Bordas, Separadores, Fundo de Campos de Formulário | `#F3F4F6` |

Esta paleta será configurada no Tailwind CSS para garantir a consistência visual.
